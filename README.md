# CRM App

CRM App is a client-side EJS application designed to help manage client information.

## Features

 * __Client Management__: add, edit, and delete client information
 * __Organization__
 * __User Authentication__
 * __Error Handling__

## Technologies

 * Node.js
 * Express.js
 * MongoDB
 * Mongoose
 * EJS
 * Tailwind CSS
 * Helmet
 * XSS Clean
 * Rate Limiter
